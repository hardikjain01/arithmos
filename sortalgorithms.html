Sorting Algorithms

A Sorting Algorithm is used to rearrange a given array or list elements according to a comparison operator on the elements.

back

Selection
Sort

Repeatedly finding the minimum element from unsorted part and putting it at the beginning.

Bubble
Sort

Repeatedly swapping the adjacent elements if they are in wrong order.

Insertion
Sort

Works the way we sort playing cards in our hands.

Merge
Sort

Divides input array in two halves, calls itself for the two halves and then merges the two sorted halves.

Quick
Sort

Picks an element as pivot and partitions the given array around the picked pivot.

Heap
Sort

Repeatedly find the maximum element and place the maximum element at the end.

Counting
Sort

Counting the number of objects having distinct key values.

Radix
Sort

Digit by digit sort starting from least significant digit to most significant digit.
