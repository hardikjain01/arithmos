Searching Algorithms

Searching Algorithms are designed to check for an element or retrieve an element from any data structure where it is stored.

back

Linear
Search

Start from the leftmost element and traverse one by one.

Binary
Search

Search a sorted array by repeatedly dividing the search interval in half.

Jump
Search

Jumping ahead by fixed steps or skipping some elements in place of searching all elements.

Interpolation
Search

May go to different locations according to the value of the key being searched

Exponential
Search

Jumping ahead by twice the steps before and perform binary search.
